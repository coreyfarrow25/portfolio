## 🗂️ Projects

### 🎧 [Beats By Dre — Consumer Insights: Data Analytics Externship](./beats-by-dre-analytics/)
End-to-end consumer insights analysis on 5,127 Amazon wireless speaker reviews. Covers data cleaning, exploratory data analysis, TextBlob sentiment analysis, correlation matrix analysis, and competitive intelligence. 30+ visualizations across EDA, sentiment, and correlation pipelines.

**Stack:** Python · Pandas · NumPy · TextBlob · Matplotlib · Seaborn · Jupyter

---

### 📱 [BeReal — Product Management Externship](./bereal-pm-externship/)
Full product management simulation for BeReal — from competitive analysis through feature development, user testing, and a stakeholder pitch. Featured deliverable: **Mood Prompt**, a new BeReal feature designed to increase daily active engagement. Includes RICE scoring, PRD-Lite, focus group design, and survey analysis.

**Frameworks:** RICE Scoring · PRD · User Journey Mapping · Focus Group Research · Stakeholder Pitch

---

### 📊 [Introduction to Statistics with R](./intro-stat-with-r/)
R scripts from introductory statistics coursework covering multi-file data merging, descriptive statistics, loops, and weighted calculations. Datasets include Netflix subscriber/revenue data and movie ratings.

**Stack:** R · `dplyr` · `rbind` · CSV I/O

---

### 📐 [Introduction to Statistical Analysis (STAT 2120)](./intro-to-statistical-analysis/)
Python and Jupyter notebook coursework covering regression, correlation, probability, confidence intervals, and hypothesis testing. Includes a capstone General Social Survey (GSS) analysis project.

**Stack:** Python · Pandas · Statsmodels · Seaborn · Matplotlib · Jupyter
